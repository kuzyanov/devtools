# dev tools

## Build
```bash
mvn clean package
```

## Run
```bash
java -jar target/devtools-1.0-SNAPSHOT.jar
```

## Swagger UI
[http://localhost:8080/swagger-ui.html](http://localhost:8080/swagger-ui.html)

## Implemented utils:
* Convert Base64 > byte array
* Convert Base64 > JSON
* Convert Base64 > string
* Unescape HTML4
* Unescape HTML4 XML
* Decode JWT token
